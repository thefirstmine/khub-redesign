Computer Science 3

General

The course employs output-based learning and focuses on client-side web development using HTML, CSS, and JavaScript. It enables the students to produce aesthetically pleasing websites, adhering to the current web standards. The course motivates students to collaborate and participate in values social issues and learn the value of teamwork to accomplish various projects.
